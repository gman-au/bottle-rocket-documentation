# Scribzee

## What is Scribzee?

* [Scribzee web site](https://www.my-oxford.com/au-en/scribzee/)

Scribzee is a free mobile / web app developed by Hamelin Brands, used for digital note-taking from paper notes.

The "Scribzee app" also refers to the [accompanying mobile app](https://play.google.com/store/apps/details?id=com.hamelin.wanapp&hl=en)
that will automatically scan Scribzee-compatible pages to a digital format.

Scribzee is another offering for 'pen-and-paper' note-takers with additional study features such as time management,
revision planning, and note organization.
